# 线程的6种状态

## 6种状态

1. New：已创建，但没启动，未执行Run方法
2. Runnable：调用了start方法，未分配资源也是此状态，Runnable可运行的，也可能是已运行中的
3. Blocked：进入synchronize修饰的代码块，仅synchronize
4. Waiting：
5. Timed Waiting
6. Terminated

## 阻塞状态

一般习惯而言，把Blocked（被阻塞）、Waiting（等待）、Timed_waiting（计时等待）都称为阻塞状态，不仅仅是Blocked
