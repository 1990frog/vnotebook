-rw-r--r--
0第一位（-）：常见文件类型（-文件、d目录、|软链接文件）
1-3一至三位（rw-）u所有者
4-6四至六位（r--）g所属组
7-9七至九位（r--）o其他人
三种权限：r读、w写、x执行

ll
drwxr-xr-x 3 cai cai
权限 所有者 所有组

chmod命令
chmod [选项] 模式 文件名
-选项
    -R 递归
-模式
    [ugoa][+-=][rwx]
    [mode=421]

chmod u+x 给u所有者赋予x执行权限
chmod g+w,o+w
chmod u-x 给u所有者删除x执行权限
chmod u=rwx 给u所有者赋予rwx权限
chmod a=rwx 给所有人赋予rwx权限
