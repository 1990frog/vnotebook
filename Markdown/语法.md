# 语法

##标题

`
\# 这是一级标题
\## 这是二级标题
\### 这是三级标题
\#### 这是四级标题
\##### 这是五级标题
\###### 这是六级标题
`

##字体
`**这是加粗的文字**
*这是倾斜的文字*`
***这是斜体加粗的文字***
~~这是加删除线的文字~~`

###效果
**这是加粗的文字**
*这是倾斜的文字*`
***这是斜体加粗的文字***
~~这是加删除线的文字~~

@RequestMapping(value = "/ds/reportform/instorageforms", method = {RequestMethod.POST})
    public Result queryDsInstorageMsgForReportForms(@RequestBody DsInstorageReportParam param){
        PageUtils<Map<String,Object>> page = new PageUtils<>();
        page.setPageSize(param.getPageSize());
        page.setCurrentPage(param.getCurrentPage());
        if(param.getReportType().equalsIgnoreCase("1") || param.getReportType().equalsIgnoreCase("2") || param.getReportType().equalsIgnoreCase("3")){
            return success(dsReportformService.queryDsInstorageReportMsg_Part1(page,param));
        }else {
            return success(dsReportformService.queryDsInstorageReportMsg_Part2(page,param));
        }
    }
