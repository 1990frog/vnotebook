# 优先队列
普通队列：先进先出；后进后出
优先队列：出队顺序和入队顺序无关；和优先级相关
```java
Interface Queue<E>
void enqueue(E)
E dequeue()
E getFront()
int getSize()
boolean isEmpty()

PriorityQueue<E> implement Queue<E>
```
