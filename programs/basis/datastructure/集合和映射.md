# 映射
map、dict
存储（键、值）数据对的数据结构（key、value）
根据键（key），寻找值（value）

```java
Map<K,V>
void add(K,V)
V remove(K)
boolean contains(K)
V get(K)
void set(K,V)
int getSize()
boolean isEmpty()
```
