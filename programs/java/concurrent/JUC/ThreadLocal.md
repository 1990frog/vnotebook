# 两大使用场景
+ 经典场景1：每个线程需要一个独享的对象（通常是工具类【非线程安全的】，典型需要使用的类有SimpleDateFormat和Random）
+ 经典常见2：每个线程内需要保存全局变量（例如在拦截器中获取用户信息），可以让不同方法直接使用，避免参数传递的麻烦

两个出发点：
1.线程不安全的类
2.避免参数传递的麻烦
